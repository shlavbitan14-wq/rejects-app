# מערכת דוחות מקצועית — בנייה וחשמל (אפליקציית דסקטופ Windows)

אפליקציה מקומית לבניית דוחות מקצועיים בתחומי **בקרת בנייה** ו**ייעוץ/בדיקות חשמל**,
עם מנוע מבוסס-בלוקים, ריבוי סוגי דוחות, טמפלייטים, עריכה חיה, חתימות וייצוא ל-PDF.

## יכולות
- **10 סוגי דוחות**:
  - בנייה — רג׳קטים שוטף · פרוטוקול מסירת דירה · בדק לפני מסירה · דוח שנת בדק · חוות דעת הנדסית.
  - חשמל — תעודת בדיקת מתקן חשמל · חוות דעת מומחה · תכנון/ייעוץ · סקר בטיחות · תרמוגרפיה.
- **50 טמפלייטים מובנים** (5 לכל סוג) + יצירת טמפלייטים מותאמים אישית.
- **עריכה מלאה תמיד** — הוספה/הסרה/הזזה של בלוקים, וכל שדה ניתן לעריכה ישירה.
- בלוקים: פרטי זיהוי, כותרת, טקסט, רג׳קטים, טבלת מדידות, רשימת בדיקה, כתב כמויות,
  חישובי עומסים, תרמוגרפיה, סיכום אוטומטי, הצהרה משפטית, חתימות.
- **עיצוב מקצועי מאופק** — פלטות צבעים נפרדות לפי קטגוריה, טיפוגרפיה עברית RTL,
  טבלאות מדידה, ומספרי עמודים ב-PDF.
- שמירה אוטומטית מקומית, ייצוא/ייבוא דוחות (‎.qcreport‎) וטמפלייטים (‎.qctpl‎).

## דרישות
- Node.js גרסה 18 ומעלה (להורדה: https://nodejs.org)

## הרצה במצב פיתוח
```bash
npm install
npm start
```

## בניית קובץ התקנה ל-Windows (.exe)
```bash
npm install
npm run build
```
קובץ ההתקנה ייווצר בתיקייה `dist/`. אין צורך ב-Node.js במחשב היעד.

## גרסה ניידת
```bash
npm run build:portable
```

## גרסת דפדפן (web)
`src/` הוא מקור-האמת. לסנכרון עותק להרצה בדפדפן:
```bash
npm run sync:web
```
ואז פתיחת `web/index.html` (משתמש ב-localStorage במקום קבצים).

## איך זה עובד
- **PDF**: נוצר ממנוע ההדפסה של Electron — נראה בדיוק כמו על המסך, כולל עברית,
  לוגו, תמונות ומספרי עמודים.
- **טמפלייטים**: נשמרים מקומית; ניתן לייצא ל-‎.qctpl‎ ולהעביר בין מחשבים.
- **תאימות אחורה**: דוחות רג׳קטים מהגרסה הישנה נטענים ומומרים אוטומטית.

## מבנה הקבצים
```
rejects-app/
├─ package.json
├─ scripts/
│  ├─ sync-web.js      העתקת src/ → web/
│  └─ serve.js         שרת סטטי לתצוגה מקדימה
├─ src/
│  ├─ main.js          תהליך ראשי: חלון, PDF (מספרי עמודים), אחסון
│  ├─ preload.js       גשר API מאובטח
│  ├─ index.html       שלד הממשק
│  ├─ styles.css       עיצוב (צ'רום + מסמך)
│  ├─ blocks.js        מנוע בלוקים (render/collect)
│  ├─ app.js           לוגיקת המערכת
│  └─ data/
│     ├─ domain.js     קבועי דומיין (פלטות, תחומים, ספי מדידה, טקסטים)
│     ├─ reportTypes.js 10 סכמות סוגי דוח
│     └─ templates.js  50 טמפלייטים מובנים
└─ web/                עותק להרצה בדפדפן (נוצר ע״י sync:web)
```
